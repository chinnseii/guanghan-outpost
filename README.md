# Guanghan Outpost / 广寒前哨

一个用于逐步迭代的 Godot 4 月球基地农业生存原型。

## 当前版本：V0.2 基地建造

当前重点是把 V0.1 的固定前哨，推进成可以逐步扩展的月球基地。

已包含：

- 星露谷式俯视移动
- 广寒前哨基础地图
- 电力、氧气、水、食物、维修件、设备完整度
- 每日资源结算
- 土豆、藻类、菌菇三种作物
- 温室种植与作物成长
- 月尘导致太阳能效率下降
- 制氧与水回收设备磨损
- 每 7 天一次地球补给选择
- 第 14 天开始的月夜压力
- 可建造模块：太阳能阵列、电池舱、小型温室、生命维持舱、维修工作台
- 模块数量会影响每日资源结算

## 运行方式

1. 安装 Godot 4.x。
2. 打开 Godot Project Manager。
3. 导入本文件夹里的 `project.godot`。
4. 点击运行。

也可以双击：

```text
launch_godot.bat
```

## 操作

- `WASD` / 方向键：移动
- `E`：与附近设施交互；建造模式下用于放置模块
- `N`：进入下一天
- `B`：打开/关闭建造模式
- `Esc`：取消建造
- 底部作物按钮：选择温室作物
- 底部建造按钮：选择要建造的基地模块

## 当前玩法目标

撑过 30 天。玩家需要在短期生存和长期建设之间取舍：

- 食物、水、氧气要够
- 太阳能板要清洁
- 生命维持设备要维修
- 第 7、14、21、28 天补给到达时选择合适货单
- 第 14 天后进入月夜，太阳能归零，电力压力上升
- 用维修件和电力扩建基地，让温室、能源和生命维持能力逐步提升
