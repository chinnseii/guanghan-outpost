# Guanghan Outpost / 广寒前哨

一个用于迭代的 Godot 4 月球基地农业生存原型。

## 当前版本

这是第一版可玩骨架，重点验证：

- 星露谷式俯视移动
- 月球基地小地图
- 电力、氧气、水、食物、维修件、设备完整度
- 温室播种与作物成长
- 太阳能板月尘清洁
- 制氧/水回收设备维修
- 每 7 天一次地球补给选择
- 第 14 天开始的月夜压力
- 中国近未来探月氛围：广寒前哨、玉兔工程车、地球补给

## 运行方式

1. 安装 Godot 4.x。
2. 打开 Godot Project Manager。
3. 选择 `Import`，导入本文件夹里的 `project.godot`。
4. 点击运行。

## 操作

- `WASD` / 方向键：移动
- `E`：与附近设施交互
- `N`：进入下一天
- 底部作物按钮：选择要播种的作物

## 第一版目标

撑过 30 天。玩家需要在短期生存和长期建设之间取舍：

- 食物、水、氧气要够
- 太阳能板要清洁
- 生命维持设备要维修
- 第 7、14、21、28 天补给到达时选择合适货单
- 第 14 天后进入月夜，太阳能归零，电力压力上升
