# Sprint 08.7: Editor Playable Demo Packaging

Status: implemented for review

## Decision

Sprint 08.7 is an editor-playable internal demo only.

Windows export, installer work, build folders, and exported-build documentation are deferred.

## Demo Goal

The owner can press Run in the Godot editor and start from the normal title screen path:

Main Menu -> Application -> Qualification Review -> National Training -> Mission Assignment -> Lunar Arrival -> Old Base Entry -> Last Plant Rescue -> Day 02 Routine -> Day 03-07 Week One Routine -> First Week End -> Phase 02 Placeholder -> Return to Main Menu.

The normal path must not require Dev / Debug entries.

## Implemented

- Main menu version label updated to `v0.5-editor-playtest`.
- Main menu now shows editor-playtest input hints:
  - 移动：WASD / 方向键
  - 交互：E / Enter
  - 返回 / 取消：Esc
  - 开发菜单：F12
- `开始新驻留` starts a clean run.
- If progress exists, `开始新驻留` shows a confirmation dialog before clearing demo progress.
- `继续驻留` includes application progress, training progress, old-base progress, and legacy save slots.
- If no progress exists, `继续驻留` remains disabled.
- Dev / Debug entry remains visually separated from normal play.
- Dev menu includes `Dev Only: Reset Demo Progress`.
- Phase 02 placeholder remains the demo end and does not start Sprint 09 gameplay.

## Deferred

- Installer
- Export preset work
- `builds/windows`
- Exported build README
