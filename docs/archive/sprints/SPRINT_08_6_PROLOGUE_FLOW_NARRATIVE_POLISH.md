# Sprint 08.6: Prologue Flow & Narrative Polish

Status: implemented for review

## Goal

Sprint 08.6 polishes the complete pre-Sprint 09 opening chain so it reads as one coherent playable prologue:

Main Menu -> Application -> Qualification Review -> National Training -> Mission Assignment -> Lunar Arrival -> Old Base Entry -> Last Plant Rescue -> Day 02 Routine -> Day 03-07 Week One Routine -> First Week End -> Phase 02 placeholder.

This sprint does not start Sprint 09 and does not add external solar array gameplay.

## Implemented

- Normal main menu path now presents the restrained official entries:
  - 开始新驻留
  - 继续驻留
  - 开发入口 / Debug
  - 退出
- Dev entries remain hidden in the dev menu and are labeled as `Dev Only`.
- Added `Dev Only: Day 07 Report Test` for weekly report regression checks.
- First Week End now closes the prologue with:
  - 第一周结束
  - 广寒前哨已恢复至最低稳定状态。
  - 第一周驻留记录已归档。
  - 下一阶段任务建议正在生成。
- Added `Phase02PlaceholderScene` as the post-prologue placeholder:
  - Phase 02：重建广寒前哨
  - 下一阶段任务：外部太阳能阵列评估
- `TrainingManager` continue flow now routes completed Week One saves to the Phase 02 placeholder.
- Mission assignment black screen text remains restrained and uses the approved 17-pioneer wording.
- Added Sprint 08.6 acceptance capture script:
  - `tools/capture_sprint08_6_acceptance.gd`
  - output folder: `docs/screenshots/sprint08_6_acceptance/`
