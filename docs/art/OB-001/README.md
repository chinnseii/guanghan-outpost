# OB-001 Old Base Core Module Target

Purpose:
This image defines the target mood and structure for the old Guanghan Outpost interior.

Use this as reference for:
- Old base core room layout
- Wall thickness and modular room frame
- Dark floor tiles
- Central console
- old power panel
- life support console
- greenhouse door
- storage lockers
- maintenance notes
- small dust / footprint details
- dim cold lighting with limited warm recovery lights

Important mood:
The old base should feel used, quiet, and carefully maintained in the past.
It should NOT feel like a horror ruin.

Implementation notes:
- Do not use the image as a flat background.
- Build the room from TileMap / modular wall sprites / separate prop nodes.
- Props should be reusable.
- Interactive objects must remain separate nodes.

Required reusable objects:
- OldPowerPanel
- PowerRestartConsole
- CentralConsole
- LifeSupportConsole
- GreenhouseDoor
- StorageLocker
- MaintenanceNote
- OldLogMarker
- WallLightPanel
- FloorDustMarks

Do not add:
- corpses
- blood
- monsters
- horror lighting
- fantasy elements
- cyberpunk neon

Acceptance target:
The current old base scene should no longer feel like a training room recolor.
It should feel like a real old lunar base that people once lived and worked in.
