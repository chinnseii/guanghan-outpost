# SOLAR-001 External Solar Array Target

Purpose:
This image defines the target look and mood for the first external engineering task after Week One.

Use this as reference for:
- lunar exterior environment
- damaged solar array
- dust-covered panels
- tilted / cracked / disconnected panel sections
- small astronaut scale
- distant old base warm lights
- Earth visible in the sky
- quiet engineering isolation

Mood:
This is not an action scene.
This is the first real external engineering problem.
The player should feel small, isolated, and responsible.

Implementation notes:
- Do not make it look like a combat arena.
- Do not add enemies.
- Do not add explosions.
- Do not make the solar panels too futuristic or shiny.
- Damaged solar panels should be readable as repair targets.

Required reusable objects:
- SolarPanelIntact
- SolarPanelDamaged
- SolarPanelDusty
- SolarPanelTilted
- CableDisconnected
- RepairMarker
- LunarRockSmall
- DistantBaseLight
- EarthSkyElement

Future gameplay link:
This scene will support Sprint 09 / Sprint 10:
- inspect damaged solar array
- identify repair needs
- later restore higher power level

Acceptance target:
The player should understand:
To expand the base, power must be restored first.
