# GH-002 Old Greenhouse Stable Target

Purpose:
This image defines the target look of the old greenhouse after minimal restoration.

Use this as reference for:
- same greenhouse layout as GH-001
- central plant chamber restored to minimal stable condition
- soft grow light on the last plant
- plant slightly greener and more upright
- surrounding greenhouse still mostly dark
- no full recovery yet

Important:
GH-002 is NOT a lush greenhouse.
It is not a victory state.
It is only the first stable state.

Visual difference from GH-001:
- central grow light is on
- plant is slightly greener
- leaves recover slightly
- monitor status changes from Critical to Stable
- small warm / green life accent appears
- surrounding racks remain dead or empty

Do not add:
- flowers
- fruits
- rapid growth
- fantasy light
- celebration effects
- full farming system

Required reusable objects:
- LastPlantStableSprite
- GrowLightOn
- PlantMonitorStable
- CentralPlantChamberStable
- SubtleGreenLightArea

Acceptance target:
The player should feel quiet relief:
The plant is not healed, but it is no longer dying.
